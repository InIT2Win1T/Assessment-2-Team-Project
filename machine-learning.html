Machine Learning

 

Machine learning is a relatively new field in the IT industry that is based around the concept of artificial intelligence. Machine learning is designed around the concept of using computational algorithms to create empirical data sets and data models. How this is achieved is from immense amounts of data collected by companies like google, microsoft, facebook etc to generate useful models of data.

By collecting this data and analyzing it and the associated datasets within it, machine learning algorithms are able to predict future trends relating to the data set, understand the underlying phenomena that created the data set and predict anomalies within the data set. In doing this, computers are able to understand the human interactions that created this data and the impact upon humans on the cyber world itself.

 

Machine learning covers multiple different fields of learning within it. These include, but are not limited to;

 

  • Computer Science
  • Mathematics
  • Statistics
  • Artificial intelligence
  • Data mining
  • Deep learning
  • Data science
  • Natural language processing

 

Machine learning is also generally split into 2 specific categories or types, Supervised learning and unsupervised learning, with 2 sub categories; semi supervised and reinforcement learning.

 

 

Supervised Learning

 

This basic description of this method is to teach the machine by using examples. This entails exposing the machine to massive amounts of labelled data sets, which can be time consuming. By exposing the machine to this massive amount of data, over time the machine is able to recognize more and more of this data and correlate it to certain inputs, i.e. images. Generally, as this process is massive in scope and rather laborious, companies such as google and amazon use crowd funded services to spread the labour across the world. Facebook has a slightly different approach, using publicly available data to train their system.

 

Unsupervised Learning

 

Unsupervised learning, contrary to supervised learning, uses algorithms instead of people to match data to emerging patterns. This method involves the machine trying to spot similarities within the data set, but not try to spot specific types of data. This is a more broad spectrum approach.

 

Semi-supervised learning

 

Semi-supervised learning is an amalgamation of both supervised and unsupervised learning. By giving the machine a certain amount of labeled data and a much larger amount of unlabelled data, the aim of this is to use these data sets to teach the machine to use the labelled data sets to learn and in turn, be able to then label the unlabelled data based upon what it learnt from the labelled data sets. This is called pseudo-labelling. If this learning model becomes as successful as the supervised learning model, it would eliminate the need for crowdfunding services and greatly reduce costs and manpower required.

 

Reinforcement learning

 

Reinforcement learning within these machines is identical to what it would be within humans. By repeating a task over and over until the task is completed successfully, the machine is able to learn from each failed attempt until the task is completed successfully.

 

 

 

What does the future look like?

 

As the uptake of machine learning grows, so does the usability of these systems. From targeted advertising, to predictive weather models, machine learning has become a fundamental technology in our daily lives.

 

Companies like Google, Microsoft, amazon and almost all ISPâ€™s (Internet Service Providers) use these systems for almost every aspect of their operations. Car manufacturers are also starting to look to this technology for their driverless cars, hospitals are using it for speech and health monitoring tools, facial recognition used by law enforcement agencies and even by IoT designers (Samsung and Apple facial recognition software). Deep learning is even being used to have machines learn to complete a task set to a human merely by observing the person at work and attempting to duplicate what they have done.

The impact upon humanity as a whole by this technology is both profound and at times, worrisome. There are growing concerns within society, that as we become more and more reliant upon machine learning and as we automate more and more processes or jobs that used to be specific to humans, this will cause individuals to lose jobs that are now being automated. Another concern is the security risk these systems pose if they were able to be hijacked by cyber criminals.

 

 

How will this affect individuals?

 

The impact machine learning will have on humanity as a whole and individuals is profound. There are both positives and negatives to this technology if it is not monitored and controlled successfully. With the increase in data being available to be used in machine learning, the concern that AI (Artificial intelligence) will become a reality is present. There are arguments on both sides of the field in relation to this, mainly due to military concerns as well as cyber security risks. In saying this, if this technology is implemented correctly and not taken advantage of, the benefits it offers individuals and humanity as a whole are truly immense. Health, Communication, Education, quality of life all, science all would see a positive increase in every aspect. As the ability to not just analyse data sets, but make predictive models based on past or historical data becomes more and more successful, both the positive and negative impacts this technology will have on humanity will be profound.
