Small Computing Devices

 

 

Small computing devices and single-board computers such as the Raspberry Pi and Arduino have proven to be utilized in a diverse amount of ways. Both these devices were initially created to serve educational purposes of teaching students about computing, coding, and electronics at a very affordable cost. Both these devices became popular with not only students, but also DIY enthusiasts and project builders, novices and professionals.

 

The Raspberry Pi is a small microprocessor-based computer on a single circuit board. It can be used for programming languages including Python, Java, C, C++. Although it was originally designed to promote programming in schools it also became used for robotics, camera-based security systems and more.

 

Arduino in a micro-controller board that enables the user to blink LEDs, accept button inputs, drive motors, read input data from sensors and more. Arduino allows simple plug-and-play quick programming through the use of a software called Arduino IDE. Using this software, a suer can write code, compile it and upload it to the Arduino. Another kind of small computing device is the Makey Makey. The Makey Makey is a small device which is connected to a computer, which then uses alligator clips to clip to conductive or party-conductive objects. The user can touch these objects (while also making contact with an alligator clip attached to an “earth” rail) to control the computer keyboard. The user can then create programs that react to keyboard inputs, which are actually conductive objects connected to the Makey Makey with alligator clips. (Makerspace for Education, n.d.) Project ideas include making a game controller by using play-dough or a musical instrument made with a house hold plant. The Makey Makey is used for educational purposes as well as being part of the maker movement  (Makerspace for Education, n.d.).

 

Regarding small computing devices, another important element is the Internet of Things (IoT). This refers to billions of devices that are connected to the Internet that are all collecting and sharing data. These devices include light bulbs that can be switched on with a mobile phone app or similar items in the household. Smart-watches and fitness bands may also be classified as IoT devices. A computer however, is not generally considered to be an IoT device and nor would a smart-phone. This is because the term refers to devices that are not normally expected to be connected to the Internet, but do interact with is independently of human action. Some larger objects such as jet engines can contain smaller components that are IoT devices, such as sensors  that obtain data and transmit it back to ensure correct operation. (Ranger, 2020)

 

As the prices of sensors and communications drops, more devices will be added to the IoT due to cost-effectiveness. Our living and work environments will contain more smart devices. Cities and towns can implement sensors to obtain data in real time to organize infrastructure, such as traffic lights and detecting how noisy and polluted the environment is. More companies will connect to the cloud as the growing number of IoT devices will generate extremely large amounts of data. Because the IoT creates large amounts of data, it is a significant drive of big-data analytics. Manufacturers can make improvements more rapidly by having vast amounts of data on how components act in real-world situations.

 

There are privacy concerns regard IoT, such as US intelligence warning that connected thermostats, cameras, and cookers could all be used either to spy on people, or to cause trouble if they were hacked.

 

Raspberry Pis and Arduinos, both being cheap computing devices  have a lot of offer in the future, especially since they have proven to demonstrate a vast array of usages. Raspberry Pis for example, make it easy for users to read code and are easy to learn coding on. They can be used by small businesses and companies that have a small budget. Some businesses have used them to automate web management, databases and use them as media servers. They can also be used for routers, file servers ,network ad-blockers, media centers and storage of large files for transfer. Raspberry Pi is a greener technology that makes it easy for small businesses to recycle their waste. (Williams, 2021)

 

Similarly, Arduino has uses in the future due to its versatility and low cost. Foundationally, it is an educational device that can be implemented in electronic circuits. This means it can be used in industrial projects to reduce the cost of prototyping. The Arduino is useful for IoT applications that focus on connectivity and feasibility. When combined with thingspeak, it has been a primary computing source for most IoT technology that is in use. The CEO and founder of Arduino, Massimo Banzi has stated that there is an expansion being created of Arduino’s usage in popular cloud platforms such as Amazon Web Services. Having worked in the education sector for over 10 years, I can see a possibility for Arduinos, Raspberry Pis and Makey Makeys to increase in usage, especially as I have witnesses schools begin to create STEM programs. Although I have spent most of my career working with children aged 5 and under these are unlikely to play a major role in my work. Learning experiences for this age group could be carried out with the Makey Makey, by using various materials to enable children to interact with computers. Using on-screen visual elements could potentially create a very basic interface to promote a simple understanding of coding. I personally own a Raspberry Pi which I used as a cheap computer while my older computer was slowing down and getting old and I did not have the funds to buy a new one yet. I tinkered with it also to experiment with different operating systems as well as seeing how it could be used for retro games. I would like to use a Raspberry Pi or Arduous for a project later on, potentially for course work. While such small computing devices currently have an appeal among enthusiasts, I can not see them being used by my many of my family and friends directly. I believe they will have an impact on our everyday devices and create that impact on the way we live however- especially with smart homes and smart cities gradually being more implemented.
