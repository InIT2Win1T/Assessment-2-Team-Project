Cloud computing, services and servers

 

Cloud computing is a relatively new IT technology that has had a large impact upon the technology industry as a whole on a global scale over the last decade. There are 4 major types of cloud computing services that are commercially available at this point in time.

 

Software as a Service (SaaS)

Platform as a service (PaaS)

Infrastructure as a service (IaaS)

Hardware as a service (HaaS)

 

 

Software as a service (SaaS)

 

Software as a service is a platform in which IT programs or applications are designed to run on a personal PC or business PC wherein the applications run on a cloud server or infrastructure using a “Pay as you go” methodology. The end user pays a subscription fee on a particular frequency to access these applications or programs that are running on the cloud network and are accessible through various methods like web client logins, remote access clients or applications that login to the cloud infrastructure. One of the main components of this type of service is that the end user does not have access or control of the cloud infrastructure or the underlying programs, they merely access them to use. At the moment, there are many different companies that offer this type of service (Google, Amazon, Microsoft) and this is only set to increase exponentially over the next few years.

 

 

Platform as a service (PaaS)

 

Platform as a service is similar to software as a service, but the difference in this model is that the applications or programs are able to be developed by the end user using the language of the platform provider. What this means is that the end user is able to make, modify and distribute applications on the platform for other users to have access to, using the service provider's chosen programming language to do so.

 

Android (Google) and IoS (Apple) are perfect examples of this, wherein a party/s develops a program or application to be released on these platforms using the programming language that the platform desires. Just like the software as a service method, the underlying infrastructure of this service is not controlled by the end user.

 

 

Infrastructure as a service (IaaS)

 

Infrastructure as a service is vastly different to the above 2 models in that this model allows end users to purchase or use cloud networking infrastructure to deploy their applications. This includes using cloud servers virtual machines to host applications and allows users to control the underlying infrastructure. Users will still use a “pay as you go” model but instead of paying a subscription for applications, they will instead pay a subscription to rent out virtual machines or servers from the IaaS providers. This will enable the user to have a much larger amount of control over this service as it includes control over the operating system or architecture as well as control over the applications or programs that will be deployed. This does come at a much larger cost and requires the user to have a much more in depth understanding of programming languages as well as the architecture that will be used to deploy this. Users are also responsible for maintenance of all software within this model as well as patching and updating it. Google, amazon, microsoft are some of the major companies that offer this kind of service.

 

 

Hardware as a service (HaaS)

 

Hardware as a service (HaaS) is very similar to Infrastructure as a Service (IaaS) with one major difference. This model is aimed at a pay as you go model of renting out full data centres to host the full service required. This service is still in its infancy and is extremely costly. Mostly aimed at middle to large businesses and includes all of the above mentioned services in it. The main benefit of this to businesses is that the business does not need to build its own infrastructure, but can instead rent it out from a 3rd party provider. Although this may be advantageous to enterprise users, this model requires the business to maintain, control and manage all upkeep to do with the service, including patching and updating all software, applications and maintenance of the infrastructure itself. The only thing that is not required is the physical control over said infrastructure.

 

 

What does the future look like?

 

WIth the evolution of cloud services and technology overall, the impact upon individuals and businesses daily operations is massive. Cloud computing enables complex and often extremely resource heavy services or applications to be offloaded onto the cloud network, enabling everyone to be able to access high powered servers or services to run applications on devices that would normally not be able to do this. From mobile device services to PC applications, by having a cloud service taking care of all of the hardware requirements, things that used to be only available to enterprise users or the rich and wealthy are now accessible by almost everyone with an internet connection.

 

There is one downside to this though, with more and more people and businesses opting into this service, security risks are starting to become more and more apparent. If a business has all its data and IT infrastructure hosted on a cloud service and this service is compromised, then the impact upon that business and its clients is massive. Also, the service provided by the cloud network provider is almost wholly dependent upon the businesses internet connection, which means that businesses and individuals that have a weak or poor internet connection cannot take full advantage of the cloud service provided as bandwidth capabilities completely control how functional, beneficial and effective the cloud services are.

 

How will this affect individuals?

 

Cloud services will have a large and lasting impact upon everyone that has any interaction with technology. By enabling users to access applications and infrastructure that they normally wouldn’t be able to access due to technical and hardware limitations, the impact upon quality of life for the end user is drastically increased. From health care applications to banking and finance, to entertainment services, there are many and varied positive outcomes from the integration of these kinds of services into our daily lives.
